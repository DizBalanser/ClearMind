\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}
This thesis presents the design and implementation of ClearMind, a Personal Digital Twin Assistant for productivity and self-management. The system combines a conversational web interface, large-language-model reasoning, user-specific long-term memory, and modular productivity services to transform unstructured thoughts into actionable tasks, schedules, reflections, and personal context updates. The work addresses the practical problem that existing productivity tools require continuous manual organization and rarely adapt to the user's habits, constraints, and goals.

The implemented prototype follows a layered architecture: a React and TypeScript frontend, a FastAPI backend, SQLAlchemy persistence, and a service layer that routes user input to specialized agents for brain-dump processing, scheduling, reflection, planning, graph analysis, and profile memory management. The thesis documents the requirements, design decisions, implementation details, validation strategy, and evaluation results of the prototype. The final system demonstrates end-to-end operation across authentication, chat orchestration, life database management, schedule export, knowledge graph exploration, profile memory curation, dashboard analytics, API documentation, and automated test coverage.
